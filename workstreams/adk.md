## Description

A developer toolkit for building and deploying agents on Kagenti. Includes a CLI, Python and TypeScript SDKs, and a local development environment so you can develop and test against Kagenti services without needing an existing cluster or Kubernetes expertise.

With one CLI install command, you get a full local Kagenti-backed environment running in a sandboxed VM. Focus areas: first-time user experience — install, platform setup, building and running agents.
