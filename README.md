## About

This repo powers the Ecosystem Guide site.

- `content/` → source content
- `docs/` → generated site (GitHub Pages)

> ⚠️ Edit `content/` only — `docs/` is auto-generated.

## Build & Preview

1. Make changes in `content/`
2. Install dependencies (one-time): `pip install mkdocs-material`
3. Build the site: `mkdocs build`
4. Commit & push both `content/` and `docs/`

Preview locally: 
```
mkdocs serve
```
Open [http://127.0.0.1:8000/](http://127.0.0.1:8000/) - live reload enabled.
